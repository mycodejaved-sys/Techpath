First Year CSE Roadmap

Your guide to the first step of B.Tech CSE journey

Step 1: Programming Basics

  • Learn C language (variables, loops, functions)
  • Understand problem-solving techniques
  • Practice simple coding problems (patterns, arrays, strings)

Step 2: Core Subjects

  • Mathematics (Calculus, Linear Algebra)
  • Digital Logic & Computer Organization
  • Engineering Physics & Chemistry basics

Step 3: Tools & Skills

  • Learn Git & GitHub basics
  • Basic Linux commands
  • MS Excel, Word, PowerPoint

Step 4: Development Basics

  • Start HTML & CSS
  • Build a personal portfolio website
  • Understand how the Internet works
